MW OSD

Open source OSD for MultiWii - Baseflight - Cleanflight - Harikiri flight controllers - or a standalone GPS based OSD

Welcome to MWOSD.

MWOSD is an Multipurpose Open Source OSD software for use primarily with UAV's and with support for the following popular flight controllers:

MULTIWII BASEFLIGHT CLEANFLIGHT HARIKIRI FIXEDWING

MWOSD also provides support a number of other configurations:

Traditional GPS based OSD with full GPS info - Altitude , direction distance etc. Traditional standalone basic OSD - basic volts / amps / RSSI Groundstation OSD for GS voltage / RSSI and timer

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
