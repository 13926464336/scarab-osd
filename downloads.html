MWOSD is an Multipurpose Open Source OSD software for use primarily with UAV's and with support for traditional GPS based OSD functionality and also significantly enhanced direct support for a number of popular flight controllers:

  • MULTIWII
  • BASEFLIGHT
  • CLEANFLIGHT
  • HARIKIRI
  • FIXEDWING

    • MWOSD also provides support a number of other configurations:

  • Traditional GPS based OSD with full GPS info - Altitude , direction distance etc.
  • Traditional standalone basic OSD - basic volts / amps / RSSI
  • Groundstation OSD for GS voltage / RSSI and timer
